SSL certificate operations
--------------------------

.. contents::
   :depth: 1
   :local:

SSL uses certificates between a server and a client, such as a Web server and an Internet browser, to secure and protect the transmitted data.

This section describes the SSL certificate operation in the Rackspace CDN API. 
The SSL certificate-related resources of the SSL Certificate API were used.

Access to this endpoint should be controlled via an RBAC system such as Repose.
